# Review Report

Date: 2026-02-14
Scope: 메타 시스템 2회차(UI 연결) 정적 리뷰

## Findings

1. High - `scenes/main.tscn`, `scripts/ui/main_menu.gd`
- 메타 서비스 접근 UI가 없어 신규 시스템을 수동 검증할 수 없었다.
- 조치: `Meta Lobby` 버튼/씬 전환 핸들러 추가.

2. Medium - `scenes/ui/meta_lobby.tscn`, `scripts/ui/meta_lobby.gd`
- 메타 기능은 구현되어도 호출 오케스트레이션이 없으면 통합 결함을 조기에 발견하기 어렵다.
- 조치: 로그인/상점/채팅/친구/파티/길드/스킬/우편/스냅샷 액션 버튼과 출력 로그 연결.

3. Medium - `docs/plans/features/ui/feature_ui_meta_lobby.md`
- UI 기획 규칙(화면 목표/흐름/컴포넌트/예외/QA)을 충족하는 문서가 없었다.
- 조치: 흐름도+컴포넌트 표+수용 기준을 추가해 계획-구현 추적성 확보.
