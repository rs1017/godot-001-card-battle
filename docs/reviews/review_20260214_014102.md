# Review Report

Date: 2026-02-14
Scope: 메타/소셜 시스템 신규 구현 전후 정적 리뷰

## Findings

1. High - `scripts/autoload/event_bus.gd`
- 기존 EventBus는 전투 이벤트만 정의되어 있어 메타 시스템 이벤트 계약이 부재했다.
- 영향: 로그인/상점/우편/채팅/친구/길드 흐름에서 신호 기반 결합 불가.
- 조치: 메타 시스템용 시그널 15종 추가.

2. High - `project.godot`
- 신규 서비스 오토로드 미등록 상태에서는 시스템 코드가 있어도 런타임에서 접근 불가.
- 영향: UI/디버그에서 서비스 호출 시 즉시 실패.
- 조치: Auth/Shop/Inventory/Chat/Social/Trigger/Skill/Mail/Log 오토로드 등록.

3. Medium - 신규 서비스 전반
- 시그널 호출을 함수처럼 호출하면 런타임 에러 가능성이 높음.
- 영향: 요청/결과 이벤트 누락 또는 크래시.
- 조치: `.emit(...)` 방식으로 통일.

## Residual Risks
- 런타임 QA(씬 실제 플레이)는 사용자 요청에 따라 미실행 상태.
- 저장/서버 연동은 아직 메모리 기반이라 재시작 시 상태가 유지되지 않음.
